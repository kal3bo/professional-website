# Professional Portfolio Website

This project is a portfolio website for a game software engineer. It uses React and Vite to deliver a fast, responsive single-page experience with dynamic sections for Home, Education, and Contact.

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Project structure

- `src/App.jsx` — main application layout and portfolio sections
- `src/App.css` — custom styling for the landing page
- `src/index.css` — global theme and base styles

## Notes
